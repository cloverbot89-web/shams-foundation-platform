# Shams Foundation Collaboration Platform

A web-based workspace for coordinating fundraising, outreach, and program activities across a distributed team.

## Overview

The Shams Foundation needs a way for volunteers, staff, and coordinators to work together on a shared mission without friction. This platform provides:

- **Simple Sign-In:** One-click Google OAuth authentication
- **Task Tracking:** Kanban boards and calendar views for discovering what needs to be done
- **Progress Reporting:** Easy ways to report accomplishments and blockers
- **Fundraising Metrics:** Track campaigns, donations, and donor relationships
- **File Sharing:** Drag-and-drop document uploads and collaboration
- **AI Chat Bot:** Ask questions about the work happening across the team
- **Real-Time Collaboration:** Comments, mentions, and notifications

## User Roles

- **Admin:** Full access, user management
- **Coordinator:** Create/manage tasks, view all activity
- **Contributor:** Create own tasks, report progress
- **Viewer:** Read-only access
